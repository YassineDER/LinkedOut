Configurer une application d'authentification

  • Téléchargez une application d'authentification. Nous recommandons Google Authenticator pour Android ou Google Authenticator pour iOS.
  • Ouvrez l'application et appuyez sur l'icône "+".
  • Scannez le code QR affiché ci-dessous.

Ce code QR est visible une seule fois. Assurez-vous de le scanner avant de continuer.
